Understanding the summon system in All Firing is crucial for making informed decisions on where to spend your hard-earned currency. The game features distinct banners, specific drop rates, and a pity system that guarantees character acquisition over time.

<Callout type="warning" title="Beware of 50/50s">
The rate-up banner operates on a 50/50 system. While the base 6-star drop rate is 1.2%, there's a 50% chance that any 6-star you pull might not be the featured rate-up character. Plan your pulls accordingly.
</Callout>

### Drop Rates and Pity System

All Firing utilizes a token-based pity system for both its rate-up and standard banners. Each summon grants you a specific token for that banner. Accumulating 80 tokens allows you to directly exchange them for a 6-star character from the respective banner's pool.

| Banner Type     | 6-Star Base Drop Rate | Pity Mechanic                                   | Exchange Cost |
| :-------------- | :-------------------- | :---------------------------------------------- | :------------ |
| **Rate-Up Banner** | 1.2% (50/50 for featured) | 1 Rate-Up Token per pull                        | 80 Rate-Up Tokens |
| **Standard Banner** | 1.2%                  | 1 Standard Token per pull                       | 80 Standard Tokens |

### Character Rarity Tiers

Characters in All Firing are categorized by star ratings, with 6-stars being the highest rarity. During summons, gold animations typically indicate a 6-star pull, while purple signifies a 5-star.

| Rarity | Visual Indicator | Notes                                    |
| :----- | :--------------- | :--------------------------------------- |
| **6-Star** | Gold animation   | Highest rarity, primary reroll target    |
| **5-Star** | Purple animation | Equivalent to SR in other gachas, useful |
