| 役割               | 説明                                       | 例となる貢献                   |
| :----------------- | :----------------------------------------- | :----------------------------- |
| **アサルト**       | 敵に大ダメージを与えることに重点を置きます。 | メインDPS、高バーストダメージ      |
| **サポート**       | 味方にバフ、回復、デバフを提供します。     | 回復、シールド、敵の弱体化         |
| **連携攻撃**       | 他のキャラクターと連携して複合攻撃を行います。 | チェーン攻撃、コンボ延長           |
| **タンク**         | ダメージを吸収し、前線の味方を守ります。   | 挑発、ダメージ軽減、高HP           |
| **戦術/戦略**      | ユニークなメカニクスや戦場制御を利用します。 | 環境操作、ユニークなバフ           |
| **ユーティリティ** | さまざまな状況に対応する汎用スキルを提供します。 | 小さなバフ、デバフ、わずかなダメージ |

### キャラクターの成長と強化

**All Firingのキャラクター**の潜在能力を最大限に引き出すには、さまざまなシステムを通じて成長に投資する必要があります。これには、レベルアップ、才能のアップグレード、使い魔の装備、レリックと刻印の活用が含まれます。

<Steps>
  <Step num="1" title="キャラクターのレベルアップ">
    戦闘やストーリーの進行を通じて経験値を獲得します。経験値素材を使用してキャラクターレベルを迅速に上げ、より高いステータスと新しい才能へのアクセスを解除します。
  </Step>
  <Step num="2" title="タレントツリーのアップグレード">
    各キャラクターには独自のタレントツリーがあります。素材（サンクタムから入手可能）を使用してアビリティをアップグレードし、戦闘効果を高め、新しいスキルを解除します。
  </Step>
  <Step num="3" title="使い魔の装備">
    使い魔はペットとして機能し、戦闘をサポートします。**同じ属性タイプ**の使い魔とキャラクターをペアリングすることで、ダメージを最大化し、属性の優位性を得ることが重要です。例えば、氷の使い魔を氷のキャラクターとペアリングします。
  </Step>
  <Step num="4" title="レリック（装備）の入手">
    レリックはキャラクターの装備であり、ステータスとセットボーナスを提供します。常にキャラクターの属性に合ったレリックセットを目指し、風のキャラクターの風ダメージ増加など、強力なボーナスを活性化させます。
  </Step>
  <Step num="5" title="刻印（覚醒）">
    ガチャシステムから重複するキャラクターを入手した場合、それらを刻印に使用できます。これらは、戦闘開始時に自動的に強度共鳴スタックを獲得するなど、キャラクターの力をさらに高める重要なボーナスを提供します。
  </Step>
</Steps>

<Callout type="success" title="スマートな召喚">
すぐに召喚したくなるかもしれませんが、プレミアム通貨（スターダストとインフィニットスターダスト）を、より強力なキャラクターや本当に欲しいバナーのために温存することは、強力なキャラクターを獲得するためのより効果的な長期戦略となる可能性があります。
</Callout>

<Section icon={Target} title="キャラクター切り替えによる戦闘の最適化">

All Firingのサイドスクロールアクション戦闘は、シームレスなキャラクター切り替えによって活かされます。このメカニクスをマスターすることは、最大のダメージを与え、攻撃を回避し、壮大なコンボを作成するために不可欠です。

<Callout type="info">
戦闘システムはキャラクターの入れ替えを重視しています。キャラクターが右隅で光っているときは、強力なコンボ攻撃のためにそれらを連鎖させる合図です。これは、追加ダメージのためにキャラクターを入れ替える他のアクションRPGと同様です。
</Callout>

### 効果的な切り替え戦略

| 戦略                   | 説明                                       | 結果                                     |
| :--------------------- | :----------------------------------------- | :--------------------------------------- |
| **チェーン攻撃**       | キャラクターが光ったときに切り替えてチェーン攻撃を起動します。 | 最大化されたダメージ、延長されたコンボ     |
| **属性バースト**       | キャラクターを回転させて属性の弱点を突きます。 | 敵の迅速な撃破、高い効率                   |
| **回避と反撃**         | 無敵フレームを持つキャラクターや素早い回避を持つキャラクターに切り替えます。 | ダメージを回避し、反撃を準備します         |
| **役割のローテーション** | サポートキャラクターでバフを与え、その後アサルトに切り替えてダメージを与えます。 | 持続的な戦闘、生存性の向上                 |
| **スタミナ管理**       | スキルがクールダウン中またはスタミナが低いキャラクターを交代させます。 | 戦闘圧力を維持し、スキル使用時間を確保します |

| 使い魔の側面 | 詳細                                       | 影響                                     |

### レリックと装備の最適化

レリックは、装備とステータスボーナスの主要な源です。戦略的な選択とアップグレードが鍵となります。

| レリックの側面  | 詳細                                       | 重要性                                   |
| :-------------- | :----------------------------------------- | :--------------------------------------- |
| **装備ステータス** | 各レリックピースはさまざまなステータスボーナスを提供します。 | キャラクターの属性を直接強化             |

| **属性の焦点**  | キャラクターの主要な属性を強化するレリックを優先します。 | 相乗効果によるパワーアップ、ビルドの専門化 |
