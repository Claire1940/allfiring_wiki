<Section icon={Swords} title="戦闘戦略とチーム切り替え">

ALLfiringの戦闘の核心は、3人のキャラクター部隊を管理することです。あなたはロゼウルとしてだけでなく、まとまりのあるユニットとしてプレイしています。**ALLfiringのロゼウル**をマスターするには、「スイッチシステム」を理解する必要があります。これにより、コンボ中にキャラクターを入れ替えて、シャドウビーストへのプレッシャーを維持することができます。

**ビデオハイライト (一般的な戦闘概要):**
- 空中コンボを延長するためのリアルタイムキャラクター切り替え。
- 敵を閉じ込めるためのボクセル環境破壊。
- 画面を最大限にクリアするための究極能力のタイミング。

| キャラクター | 属性 | 役割 / 特性 |
| :--- | :--- | :--- |
| **ロゼウル** | 雷 | 戦術情報 / サポート |
| **キリカ** | 火 | ヴァンガード / ヘビーハンマー |
| **ナエ** | 氷 | ストライカー / 刀マスター |
| **ハーフウィン** | 闇 | 影操作 |
| **フェルギア** | 風 | ハイプリーステス / ユーティリティ |

ロゼウルがフィールドにいるとき、あなたの目標は舞台を整えることです。彼女のスキルを使って敵に「導電」状態を適用します。フィールドが電化されたら、キリカのような火属性のキャラクターに切り替えて「オーバーロード」爆発を誘発したり、ナエのような氷属性のキャラクターに切り替えて敵のアーマーを破壊する「超電導」デバフを作成したりできます。

<Callout type="tip" title="戦闘のヒント">
常に敵の攻撃パターンを見てから交戦しましょう。ロゼウルの戦術情報能力は、敵の予兆をより明確に見ることができるため、あらゆるボス戦を開始するのに最適なキャラクターです。

<Section icon={Target} title="進行とレベリングガイド">

ALLfiringでのレベルアップは、初心者にとって圧倒されることがあります。最もよくある間違いは、ガチャシステムから入手したすべてのキャラクターにリソースを分散させてしまうことです。コアチームの可能性を最大限に引き出すには、厳格なアップグレードパスに従う必要があります。

<Steps>
  <Step num="1" title="コアチームの選択">
    3人のキャラクターを選び、彼らと共に進みましょう。ロゼウルはあなたの主要なユーティリティ/サポートの選択肢であるべきです。
  </Step>
  <Step num="2" title="メインストーリーの重視">
    まずストーリーチャプターを完了しましょう。これらは重要なメカニクスを解放し、序盤のエンベライトとスターダストの大部分を提供します。
  </Step>
  <Step num="3" title="ボクセルスカベンジング">
    出口に直行するだけでなく、すべてのブロックを破壊し、隅々まで探索して、ロゼウルのスキルアップグレードに必要な希少な素材を見つけましょう。
  </Step>
  <Step num="4" title="スキルの優先順位付け">
    ロゼウルの「戦術スキャン」と「サンダーチェーン」を最初にアップグレードすることに焦点を当て、チームローテーションでの彼女のユーティリティを向上させましょう。
  </Step>
</Steps>

| リソース | 主要な入手源 | 用途 |
| :--- | :--- | :--- |
| **エンベライト** | デイリークエスト / ストーリー | キャラクターの基本レベルアップ |
| **スターダスト** | ボクセル探索 | スキルツリーのアップグレード |

<Section icon={Monitor} title="MuMuPlayerでPCでプレイする方法">

ALLfiringはモバイルファーストのタイトルですが、そのボクセルプラットフォームの複雑さとリアルタイム戦闘は、PCでのプレイに最適な候補となります。MuMuPlayerを使用すると、ロゼウルのクイックスイッチコンボを実行するために不可欠な正確なキーマッピングが可能になります。

| 機能 | モバイル体験 | PC (MuMuPlayer) 体験 |
| :--- | :--- | :--- |
| **操作** | タッチスクリーン（不正確になることがある） | キーボード/マウス（非常に正確） |
| **グラフィック** | バッテリー/熱によって制限される | 最大設定、60+ FPS |
| **画面サイズ** | 小さく、UIがごちゃごちゃする可能性がある | 大画面モニター、視認性向上 |
| **マルチタスク** | 困難 | サイドでガイドを実行するのが簡単 |

**PCでのALLfiringのインストール手順:**

1.  **MuMuPlayerをダウンロード**: 公式サイトにアクセスし、WindowsまたはMac用の最新バージョンをインストールします。
2.  **Googleサインイン**: セットアップを完了してアプリセンターにアクセスします。
3.  **ALLfiringを検索**: ゲームを見つけてインストールをクリックします。お住まいの地域にない場合は、APKファイルを手動でインストールできます。
4.  **マッピングを設定**: キャラクター1、2、3のキーをアクセスしやすいボタン（1、2、3またはQ、W、Eなど）に設定します。
5.  **プレイを開始**: ホーム画面からゲームを起動し、強化されたパフォーマンスをお楽しみください。

<Callout type="success" title="パフォーマンス向上">
PCでプレイすることで、ロゼウルの雷能力の強烈な視覚効果に伴うラグが軽減され、回避を逃すことがなくなります。
</Callout>
