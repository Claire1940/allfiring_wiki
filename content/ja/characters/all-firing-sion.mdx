### スキル概要

サイオンのスキルを理解することは、彼の「オールファイアリング」戦略の基本です。各スキルが彼の耐久力、ダメージ、またはユーティリティに貢献します。

| スキル | タイプ | 主な機能 | 二次的な機能 |
| :------ | :------------ | :----------------------------------- | :---------------------------------- |
| **P - 死せる栄光** | パッシブ | 死後の戦闘 | 高ダメージ、サステイン |
| **Q - 大量虐殺** | ダメージ、CC | チャージ可能な範囲ダメージ＆ノックアップ | ウェーブクリア、ゾーンコントロール |
| **W - 魂の炉心** | シールド、ダメージ | 一時的なシールド、最大HP増加 | 範囲魔法ダメージ（シールド後） |
| **E - 殺戮の咆哮** | スロウ、物理防御減少 | 対象のスロウ、ミニオンの吹き飛ばし | チャンピオンへの物理防御減少 |
| **R - 猪突猛進** | エンゲージ、ダメージ | 長距離ダッシュ、範囲スタン＆ダメージ | マッププレッシャー、集団戦の開始 |

</Section>

適切なスキルのレベル上げは、サイオンの序盤の展開と、真の「オールファイアリング」の脅威へと成長するために不可欠です。

### スキルレベル上げの優先順位

トップサイオンにおける標準的かつ最も効果的なスキル順序は以下の通りです：

<Steps>
  <Step num="1" title="Q（大量虐殺）を最大にする">
    強力なウェーブクリア、ハラス、バーストダメージのためにQを最優先します。これによりレーンを支配し、ガンクをお膳立てできます。
  </Step>
  <Step num="2" title="W（魂の炉心）を最大にする">
    Qの次は、シールド量と基本体力の獲得量を増やすためにWを上げます。これにより、中盤の集団戦での生存能力が大幅に向上します。
  </Step>
  <Step num="3" title="E（殺戮の咆哮）を最大にする">
    Eの主な用途（スロウと防御減少）はQやWほどレベルによる伸びが大きくないため、最後に上げます。
  </Step>
  <Step num="R" title="可能な限りRを上げる">
    アルティメットスキルの猪突猛進は、レベル6、11、16で取得可能になり次第、必ずポイントを振ってください。
  </Step>
</Steps>

### 基本コンボ

これらの基本コンボをマスターすることで、サイオンのプレイの質が向上します：

-   **レーンでのハラス/トレード:** E（ミニオンまたは敵に） -> Q（最大ダメージ/ノックアップのためにチャージ） -> W（トレード/ダメージ用のシールド）。
-   **エンゲージ/集団戦:** R（猪突猛進）で敵の後衛に突っ込む -> Q（フルチャージのノックアップ） -> W（ダメージ吸収用のシールド） -> E（対象へのスロウと防御減少）。
-   **死後:** パッシブの「死せる栄光」を使用して、体力の低い敵を追い詰めるか、構造物に大きなダメージを与えます。この状態ではQとEのスキルが強化されます。

### 苦手なチャンピオン

サイオンは、高い機動力を持つチャンピオン、一貫したハラス、または彼の膨大な体力を無視できる割合ダメージを持つチャンピオンを苦手とします。

| チャンピオン | 勝率 | 理由 |
| :------------ | :------- | :------------------------------------------------- |
| **シン・ジャオ** | 44.5% | 序盤のダメージが高く、強力なデュエリストであり、Qを中断させることができる。 |
| **クイン** | 45.0% | 高い機動力、射程、一貫したハラス。 |
| **ハイマーディンガー** | 45.3% | タレットによる絶え間ないプレッシャーとゾーンコントロール。 |
| **オラフ** | 46.2% | 確定ダメージ、RによるCC無効、強力なデュエリスト。 |

<Callout type="warning">
クインやハイマーディンガーのようなチャンピオンに対しては、ハラスへの耐性を高めるために「不滅」ツリーの**息継ぎ**を取ることを検討してください。
</Callout>

### 得意なチャンピオン

サイオンは、機動力に欠ける近接チャンピオンや、彼の防御を突破する持続ダメージを持たないチャンピオン、予測可能なパターンを持つチャンピオンに対して強みを発揮します。

| チャンピオン | 勝率 | 理由 |
| :--------- | :------- | :----------------------------------------------- |
| **マスター・イー** | 59.5% | QやRでCCをかけやすく、タンクを倒すのに苦労する。 |
| **アカリ** | 57.0% | サイオンのHPを削り切るのに苦労する。 |
| **ランブル** | 56.8% | 予測可能なダメージで、サステインで上回りやすい。 |
| **ポッピー** | 56.3% | サイオンのQはポッピーにとって中断させるのが難しい場合がある。 |
