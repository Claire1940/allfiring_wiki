<Section icon={Swords} title="戦闘メカニクスとチームシナジー">

ALLfiringの戦闘はペースが速く、サイドスクロールで、素早い反射神経と戦略的思考が必要です。システムの中心は3キャラクター切り替えメカニズムで、これにより中断することなく戦闘の流れを維持できます。

- **リアルタイム切り替え**: コンボ中にキャラクターを入れ替えて、攻撃の連鎖を延長できます。
- **元素シナジー**: 特定のキャラクターは、環境や敵の弱点と相互作用する元素特性を持っています。
- **究極のアビリティ**: 各仲間は、困難なボス戦の流れを変えることができるシグネチャー技を持っています。

| 戦闘アクション | 説明 | 最適な使用法 |
| :--- | :--- | :--- |
| **軽攻撃** | 速く、低ダメージの攻撃 | コンボメーターを構築する |
| **重攻撃** | 遅く、高インパクトの攻撃 | 敵のガードを破る |
| **スイッチスキル** | キャラクター交換時に発動 | プレッシャーを維持する |
| **フレイムバースト** | 強力な範囲攻撃 | 敵の集団を一掃する |

例えば、キャラクターのエルクシスを使いこなすことは、バランスの取れたステータスと興味深いストーリーアークのため、初心者には強くお勧めします。プレイヤーは、戦闘がスムーズで、ストーリーの文法的な誤りがほとんどないため、物語の旅も戦闘と同じくらい楽しいと指摘しています。
