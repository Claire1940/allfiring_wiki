ALLfiringは、開発中のタイトル「プロメテウス」としても知られ、2026年のモバイルゲーム界に革命をもたらすタイトルです。GENMUGAME CO., LTD.は、3Dボクセルサンドボックスとハイオクタンな2Dアニメ横スクロール戦闘を融合させることで、インタラクティブかつ危険な世界を創造しました。最後の火守り人として、あなたの使命は、影に飲み込まれた世界を取り戻し、仲間と共に希望を再建することです。この総合的な「**ALLfiring攻略**」ガイドは、荒廃した土地を navigated し、複雑な戦闘メカニクスとボクセル探検をマスターするのに役立ちます。これらは、このタイトルを従来のARPGと区別する要素です。横スクロールゲームのベテランであろうと、ガチャシステムの初心者であろうと、ガーディアンと環境のシナジーを理解することが生存に不可欠です。

<Section icon={Map} title="ボクセルサンドボックスワールドの理解">

ALLfiringの世界は単なる背景ではありません。完全にインタラクティブな遊び場です。プラットフォームが静的な従来の横スクロールゲームとは異なり、ここのボクセル環境は再形成することができます。ブロックを壊して隠された道を発見したり、階層構造を登ったり、一見見えない隅に隠された宝箱を見つけたりすることができます。

| 特徴 | インタラクションの種類 | 利点 |
| :--- | :--- | :--- |
| **ボクセルブロック** | 破壊可能 | 隠された道と希少な素材を明らかにする |
| **階層プラットフォーム** | 移動可能 | 垂直方向の探索と側面攻撃を可能にする |
| **インタラクティブオブジェクト** | トリガー可能 | 秘密のドアを開けたり、環境トラップを起動したりする |
| **隠れた隙間** | 探索可能 | 高価値の戦利品とアップグレードジェムが含まれる |

このゲームでは探索が非常に報われます。メインストーリーを急いで進めると、より早くエンドゲームに到達できるかもしれませんが、中盤の進行に必要な重要な素材を逃してしまうでしょう。マップの隅々まで意図を持って設計されており、多くの場合、戻るのが大幅に楽になるショートカットが隠されています。

<Callout type="info" title="探索のヒント">
珍しいブロックの形成を見過ごさないでください。攻撃を使って破壊可能かどうかを試してみてください。ゲーム序盤の最高の武器の多くは、一見頑丈に見えるボクセル壁の裏に隠されています。
</Callout>

</Section>

<Section icon={Swords} title="戦闘をマスターするためのALLfiringのヒント">

ALLfiringの戦闘は「トライガード」システムを中心に展開されます。3体のキャラクターでチームを編成し、ワンタップで切り替えることができます。これは単なる多様性のためではなく、コアとなる生存メカニクスです。キャラクターを切り替えることで、特定のアニメーションフレームがリセットされ、クールダウンの長いアビリティを連鎖させることができます。

**コア戦闘メカニクス:**
- **通常攻撃**: 基本中の基本であり、基本的なコンボに繋がります。
- **スキルバースト**: 視覚的に派手な高ダメージアビリティで、シャドウビーストの群れを一掃できます。
- **究極アビリティ**: ゲームの流れを変える技で、ボスやエリートとの遭遇のために温存すべきです。
- **キャラクター切り替え**: **ALLfiring攻略**の中で最も重要です。攻撃的なプレッシャーを維持しながら、 incoming 攻撃を回避するためにこれを使用します。

<Steps>
  <Step num="1" title="攻撃パターンを観察する">
    突撃する前に、シャドウビーストの動きを観察してください。ほとんどの敵は、 incoming の重い一撃を知らせる明確な合図を持っています。
  </Step>
  <Step num="2" title="属性の優位性で開始する">
    敵のタイプに対して属性が強いキャラクターで戦闘を開始し、早期に状態異常を付与します。
  </Step>
  <Step num="3" title="連鎖と切り替え">
    フルコンボを実行し、スキルを発動し、すぐに次のキャラクターに切り替えて、中断することなく勢いを維持します。
  </Step>
  <Step num="4" title="究極を発動する">
    ボスがスタンしているか、体力閾値に達したら、3つの究極すべてを素早く連続して放ち、最大のバーストダメージを与えます。
  </Step>
</Steps>

<Callout type="success" title="コンボの秘密">
スキルが着弾する直前にキャラクターを切り替えると、「連携攻撃」が発動し、離脱するキャラクターがフィールドを離れる前にもう一度攻撃を行います。
</Callout>

</Section>

<Section icon={Zap} title="キャラクターの属性と役割">

ALLfiringでの成功は、チームの役割と属性のバランスをどれだけうまく取れるかにかかっています。各キャラクターはフィールドに独自の専門性をもたらし、偏ったチームは後のチャプターで見られる多様なシャドウビーストに対して苦戦するでしょう。

| キャラクター | 属性 | 役割 / 専門分野 | 主な利点 |
| :--- | :--- | :--- | :--- |
| **ロゼール** | 雷 | 戦術 / 情報 | 高速ダメージバーストと偵察 |
| **キリカ** | 火 | 先鋒 / ハンマー | 重装甲破壊と高HP |
| **ナエ** | 氷 | ストライカー / 刀 | 精密攻撃と群衆制御 |
| **ハーウィン** | 闇 | ユーティリティ / 影 | 味方へのバフと敵へのデバフ |
| **フェルギア** | 風 | ユーティリティ / 巫女 | 回復と移動速度バフ |
| **シオン** | 風 | ガーディアン / メッセンジャー | バランスの取れた防御と空中戦 |

属性システムはダイナミックです。火属性のキャラクターは爆発的なダメージに焦点を当て、氷属性のキャラクターは敵を減速させる達人です。風属性のキャラクターは柔軟性と速度を提供し、雷属性は高速のバーストウィンドウを提供し、闇属性のキャラクターは神秘的な力を使って戦場を操作します。

<Callout type="warning" title="チーム編成の注意点">
ガチャで引いたすべてのキャラクターをアップグレードするのは避けましょう。エンベライトやスターダストのようなリソースは希少です。現在のチャプターのレベルキャップに到達するまで、上位3体のガーディアンにのみ集中しましょう。
</Callout>

</Section>

<Section icon={Shield} title="MuMuPlayerでPCでプレイする">

大画面とより正確なコントロールを好むプレイヤーのために、ALLfiringはMuMuPlayerを使用してPCまたはMacでプレイできます。この設定により、カスタムキーマッピングが可能になり、高レベルのプレイで必要とされる高速なキャラクター切り替えに非常に役立ちます。

| 要件 | 推奨設定 |
| :--- | :--- |
| **エミュレータ** | MuMuPlayer 12 (最新バージョン) |
| **グラフィックモード** | Vulkan または DirectX 11 |
| **RAM割り当て** | 4GB以上 |
| **キーマッピング** | キャラクター切り替えを1、2、3にマッピングする |

**PCでのセットアップ手順:**
1. 公式サイトからMuMuPlayerをダウンロードしてインストールします。
2. Googleサインインプロセスを完了してPlayストアにアクセスします。
3. 「ALLfiring」を検索してインストールをクリックします。
4. 移動とスキル実行のためのキーボードショートカットを設定します。
