<Section icon={Sparkles} title="元素システムとファミリア">

元素システムは、**all firing グローバルローンチガイド**に戦略の層を追加します。火、氷、風、雷、闇の5つの主要な元素があります。キャラクターの元素に加えて、ゲームには「ファミリア」という、戦闘を助け、大幅なステータスブーストを提供するペットが登場します。

| 元素   | 戦闘スタイル         | 敵への効果             |
|--------|--------------------|------------------------|
| **火** | 爆発的 / 高ダメージ  | ターゲットを燃焼させ持続ダメージ |
| **氷** | 制御 / スロー      | 動きを凍結または減速させる |
| **風** | 迅速 / 柔軟        | 迅速な攻撃と高い機動性 |
| **雷** | バースト / 派手    | ターゲット間でダメージを連鎖 |
| **闇** | 神秘的 / デバフ    | 強力な状態異常         |

**ファミリアシナジーのルール:**
1. **元素を合わせる**: 風属性のキャラクターには常に風属性のファミリアを装備させましょう。これにより、ファミリアのダメージがキャラクターの元素バフに合わせてスケーリングされます。
2. **スキルペアリング**: キャラクターがストライカーの場合、防御シールドではなく攻撃的なバフを提供するファミリアを選びましょう。
3. **ベアダンピング（氷）**: イルシャのような氷属性キャラクターに優れた群衆制御を提供するトップティアの初期ゲームファミリアです。

<Section icon={Shield} title="進行とスタミナ管理">

この**all firing グローバルローンチガイド**で強調されている一般的な間違いは、スタミナ（エネルギー）の管理ミスです。このゲームでは、すべての素材収集ダンジョンでスタミナシステムが使用されており、コストが驚くほど高いです。

| アクティビティ     | スタミナコスト | 主な報酬                   |
|------------------|--------------|----------------------------|
| **ストーリーステージ** | 0 (初回クリア) | エンバーライト、スターダスト、経験値 |
| **聖域 (才能)**    | 40           | アビリティアップグレード素材 |
| **ホール (ダンジョン)** | 30-40        | レリック (装備) とゴールド |
| **ファミリアグラウンド** | 40           | ファミリアアップグレードアイテム |
| **昇天試練**       | 50           | キャラクターレベル上限突破   |

<Callout type="warning" title="スタミナに関する警告">
合計240スタミナのキャップがあり、1回のランで40のコストがかかるため、1日の許容量をわずか6回の戦闘で使い果たす可能性があります。多くのトッププレイヤーは、最初の1週間は1日に少なくとも2回スタミナを回復するためにプレミアム通貨の一部を使用することを推奨しています。
</Callout>

**装備とレリック:**
レリックはあなたのギアとして機能します。それらはセット（2ピースと4ピース）で提供されます。例えば、2ピースの風セットは風ダメージを増加させます。ミスマッチなセットを装備しないでください。セットボーナスは、初期のゲームではより高いレアリティのギアの個々の基本ステータスよりもはるかに価値があります。

</Section>
