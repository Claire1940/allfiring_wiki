<Section icon={Swords} title="戦闘メカニクスと属性">

ALLfiringの戦闘は、3人のキャラクターで構成される部隊を中心に展開されます。一度に1人のキャラクターを使用する多くのガチャゲームとは異なり、ALLfiringは絶え間ない切り替えを推奨しています。コンボ中にキャラクターアイコンをタップすると、交代するユニットが「スイッチイン」攻撃を実行し、ダメージの流れを維持し、攻撃的なプレッシャーを失うことなく迫りくるボスギミックを回避できます。

属性システムは、戦闘が単調に感じられるのを防ぐ戦略の層を追加します。5つの主要な属性があり、それぞれに特定の戦闘特性があります。

| 属性 | 戦闘スタイル | 主な利点 |
| :--- | :--- | :--- |
| **火** | 爆発的 / 重量級 | 高いバーストダメージとAOE効果 |
| **氷** | 制御 / 鈍化 | 敵を凍らせ、移動速度を低下させる |
| **風** | 迅速 / 柔軟 | 高い機動性と素早いコンボの可能性 |
| **雷** | 高速 / 派手 | 高い攻撃速度とバーストスキルチェーン |
| **闇** | 神秘的 / ユーティリティ | 強力なデバフとクラウドコントロール |

<Callout type="warning" title="リソース管理">
引いたすべてのキャラクターをレベルアップしようとしないでください。序盤のアップグレード素材は希少です。最高の3人のキャラクターを選び、レベルキャップに達するまで彼らにすべてのリソースを投入してください。
</Callout>

さらに、キャラクターは特定の役割に分けられます。バランスの取れたチームは通常、1人の主要なダメージディーラーと、ユーティリティまたは連携攻撃を提供する2つのユニットで構成されます。

- **ヴァンガード/タンク**: 高い生存性、前線に留まりダメージを吸収します。
- **ストライカー/アサルト**: 短い時間で可能な限り最高のダメージを与えることに集中します。
- **タクティカル/ストラテジー**: バフ、フィールドコントロール、または敵へのデバフを提供します。
- **ユーティリティ/ヒーラー**: チームを生き延びさせ、負のステータス効果を解除します。
- **連携攻撃**: 他のユニットがアクティブな間、オフフィールドダメージに特化します。

</Section>

<Section icon={Target} title="ステップバイステップの進行ガイド">

最初の週に効率を最大化するには、この構造化された進行パスに従ってください。これにより、プレミアム通貨を無駄にしたり、難易度の急上昇で立ち往生したりすることがなくなります。

<Steps>
  <Step num="1" title="プロローグを完了する">
    メインストーリーチャプターに完全に集中してください。これらはチュートリアルとして機能し、ガチャシステム、武器アップグレード、デイリーミッションなどのコア機能をアンロックします。
  </Step>
  <Step num="2" title="事前登録報酬を受け取る">
    メールボックスに移動し、すべてのエンバライトとスターダストを受け取ります。これらを使用して最初の召喚を行いますが、強力な初期トリオが揃ったら**ジェムを温存**してください。
  </Step>
  <Step num="3" title="6つ星セレクターを使用する">
    最初の召喚の後、セレクターを使用してチームのギャップを埋めます。強力なDPSを引いた場合はガルシアを選び、ダメージが不足している場合はローズを選びます。
  </Step>
  <Step num="4" title="コアトリオをアップグレードする">
    3人の主要ユニットにアップグレード素材を集中させます。スキルをレベルアップし、武器を強化し、6つ星使い魔を装備して戦闘力をブーストします。
  </Step>
  <Step num="5" title="デイリールーチン">
    すべてのデイリーミッションを完了し、「シャドウビースト」ファームステージをクリアして、キャラクターの覚醒に必要な素材を集めます。
  </Step>
</Steps>
