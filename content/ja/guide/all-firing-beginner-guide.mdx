### 属性の相性とキャラクターの役割

ALLfiringの戦闘システムの重要な側面は、属性の相性とキャラクターの役割です。これらの側面を相乗効果的に組み合わせたバランスの取れたチームを構築することは、より強力な敵やボスに立ち向かうために不可欠です。

| 属性      | 一般的な焦点                           | 例                                          |
| :-------- | :------------------------------------- | :------------------------------------------ |
| **火**    | 爆発的な攻撃、高いダメージ             | キリカ（ヴァンガード / ヘビーハンマースペシャリスト） |
| **氷**    | 敵の速度低下、群衆制御                 | ナエ（ストライカー / 精密カタナマスター）     |
| **風**    | 素早い攻撃、柔軟な戦闘スタイル         | フェルギア（ハイプリーステス / 教会ユーティリティ） |
| **雷**    | 迅速なダメージバースト、派手なスキル   | ロゼウル（情報ブローカー / 戦術インテル）   |
| **闇**    | 神秘的な力、強力な効果                 | ヘルフウィン（パティシエ / シャドウマニピュレーション） |

キャラクターはまた、戦場での有用性を決定する明確な役割に分類されます。

| 役割               | 説明                                             | 主要キャラクター（例） |
| :----------------- | :----------------------------------------------- | :--------------------- |
| **タンク**         | 最前線の防御者、ダメージを吸収                   | ヴァンガードタイプ     |

| **ユーティリティ** | サポート役割、回復、バフ、デバフ                 | フェルギア             |
| **連携攻撃**       | 他のキャラクターと連携して強力なコンボ攻撃を行う |                        |

### 戦闘の流れをマスターする

<Steps>
  <Step num="1" title="キャラクター切り替えを理解する">
    戦闘中、3人のアクティブなキャラクター間をシームレスに切り替えることができます。これは単なる見せかけではありません。コンボを延長し、異なるスキルセットを活用し、ダメージを回避するために不可欠です。右隅でキャラクターが光っているとき、彼らはチェーン攻撃の準備ができています。
  </Step>
  <Step num="2" title="敵のパターンを観察する">
    各シャドウビーストは明確な攻撃パターンを持っています。特にボスに対しては、積極的に行動する前にこれらのパターンを観察するために少し時間をかけましょう。いつ回避するか、いつキャラクターを切り替えるかを知ることは、生存とダメージ最大化のために非常に重要です。
  </Step>
  <Step num="3" title="スキルとアルティメットを連鎖させる">
    異なるキャラクターの組み合わせを試して、強力なスキルシナジーを見つけましょう。重要な瞬間にアルティメットアビリティのタイミングを合わせることで、困難な戦いの流れを変えることができます。
  </Step>
  <Step num="4" title="属性の利点を活用する">
    キャラクターとファミリアを同じ属性で組み合わせると、強力なシナジーが生まれます。たとえば、氷属性のキャラクターと氷属性のファミリアを組み合わせると、氷に弱い敵に対してはるかに多くのダメージを与えることができます。
  </Step>
</Steps>

<Callout type="tip" title="最適なファミリアの組み合わせ">
常に同じ属性タイプのファミリアをキャラクターに装備させるようにしましょう。これにより、ダメージ出力が安定し、脆弱な敵に対して最大化され、チームがはるかに効果的になります。
</Callout>

### キャラクターの育成と管理

キャラクターは、毎日のログイン報酬、マップクリア、クエスト完了、ダンジョンでの素材集めを通じて経験値とリソースを獲得します。これらのリソースを効率的に管理することが、迅速な進行の鍵となります。

| 進行の側面 | 説明                                                       | 最適化のヒント                                       |
| :--------- | :--------------------------------------------------------- | :--------------------------------------------------- |
| **レベルアップ** | キャラクターのステータスを上げ、新しいタレントをアンロックする | まず主要な3人のチームに集中し、リソースの分散を避ける。 |
| **タレント** | 特定のアビリティとパッシブボーナスをアップグレードする     | 主要なダメージディーラーやサポートを強化するタレントを優先する。 |
| **ファミリア** | 戦闘を支援し、属性ダメージを提供するペットの仲間           | キャラクターと同じ属性を持つファミリアを合わせる。   |
| **遺物（装備）** | ステータスとセットボーナスを持つ装備を装着する             | 特定の属性ダメージやキャラクターの役割を強化する装備セットを探す。 |
| **刻印**   | 複製キャラクターを使用して「覚醒」システム                 | 複製キャラクターを使用して強力な刻印ボーナスをアンロックする。 |

</Section>

### プレミアム通貨：スタミナ vs 召喚

新規プレイヤーにとって最大のジレンマの1つは、プレミアム通貨（スターダストなど）を召喚に使うべきか、スタミナに使うべきかということです。スタミナは、キャラクターの強化に不可欠な素材をダンジョンで集めるために使用されます。

<CardGrid cols={2}>
  <Card title="召喚に使う">
    - **長所**: 新しい強力なキャラクターやファミリアを獲得するチャンス。引く興奮。
    - **短所**: 運に左右される、重複やあまり役に立たないユニットが出る可能性。リソースがすぐに枯渇する可能性。
  </Card>
  <Card title="スタミナに使う">
    - **長所**: キャラクターアップグレードに必要な素材が確実に手に入る、既存チームの進行が速くなる。
    - **短所**: 新しいキャラクターを獲得する直接的なチャンスがない。召喚よりも面白みに欠ける。
  </Card>
</CardGrid>

<Callout type="success" title="初心者向け推奨">
最初の観察に基づくと、初心者にとっては、プレミアム通貨をスタミナ回復に優先的に使う方が有益であることが多いです。これにより、必須のアップグレード素材を安定して集めることができ、コアチームが強力な状態を保ち、より困難なコンテンツに取り組むことができます。召喚は、本当に欲しいキャラクターが登場する特定のバナーのために温存することができます。
</Callout>
