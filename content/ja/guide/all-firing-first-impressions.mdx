### 元素の親和性システム

All Firingの元素システムは、戦闘に深さと戦略の層を追加します。火、氷、風、雷、闇の5つの主要な元素があります。各元素には、他に対する特定の強みと弱みがあり、困難な遭遇のためにチーム構成を決定的にします。キャラクターをその元素に基づいた互換性のある使い魔や遺物と組み合わせることで、その効果が大幅に向上します。

| 元素     | 主な焦点                | 典型的な効果                           |
| :------- | :---------------------- | :------------------------------------- |
| **火**   | 高ダメージ、爆発        | 燃焼、範囲攻撃（AoE）ダメージ          |
| **氷**   | 群衆制御、減速          | 凍結、敵の減速                         |
| **風**   | 敏捷性、柔軟な戦闘      | 素早い攻撃、移動バフ                   |
| **雷**   | バーストダメージ、派手なスキル | スタン、高い単一ターゲットダメージ     |
| **闇**   | 神秘的な力              | デバフ、強力なユニークな効果           |

<Callout type="warning" title="元素のミスマッチ">
キャラクターの元素と矛盾する元素ダメージタイプの使い魔や遺物を装備することは避けてください。例えば、氷のキャラクターに火ダメージの使い魔を与えると、ダメージ出力が薄まり、全体的な効率が低下します。常に元素シナジーを追求しましょう。
</Callout>

### 使い魔と遺物：武器を強化する

使い魔は、戦闘をアシストし、追加のダメージやユーティリティを提供するペットとして機能します。遺物は本質的にあなたの装備であり、ステータスボーナスとセット効果を提供します。どちらもキャラクターの進行にとって重要であり、キャラクターの元素と役割を補完するように慎重に選択する必要があります。

| アイテムタイプ     | 機能                                   | 主要戦略                                     |
| :----------------- | :------------------------------------- | :------------------------------------------- |
| **使い魔**         | 戦闘をアシストし、元素ダメージを提供する | 使い魔の元素とキャラクターの元素を一致させる |
| **遺物**           | ステータスとセットボーナスを持つ装備   | キャラクターの元素/役割を強化する遺物を装備する |
| **エンブレム**     | 永続的なキャラクターアップグレード     | 重複するキャラクターのコピーを強力なボーナスに使用する |

<Callout type="success" title="エンブレムの最適化">
エンブレムは、大幅なステータスブーストとユニークな能力を提供します。ガチャシステムから重複するキャラクターを入手した場合は、エンブレムに使用してその潜在能力を最大限に引き出し、戦闘開始時の共鳴力などの強力な新しい効果を獲得しましょう。
</Callout>

### リソース管理とスタミナ

All Firingにはスタミナシステムがあり、主にダンジョンでの素材収集に使用されます。スタミナはすぐに消費されるため、その管理は進行の重要な側面です。スタミナ回復にプレミアム通貨を使うべきか、ガチャ召喚のために貯めるべきかについては、プレイヤーの間で議論があります。ダンジョンの入場料が高いことを考えると、スタミナに投資することはキャラクターのパワーアップを加速させることができます。

| リソースタイプ       | 主な用途                          | 取得方法                                 | 戦略的用途                                |
| :------------------- | :-------------------------------- | :--------------------------------------- | :---------------------------------------- |
| **スタミナ**         | ダンジョン周回、素材収集          | 時間経過による回復、プレミアム通貨       | キャラクター昇格と才能に不可欠            |
| **スターダスト**     | ガチャ召喚（限定バナー）          | イベント、デイリーミッション、プレミアム通貨 | 目的の限定キャラクターのために貯める      |
| **無限スターダスト** | ガチャ召喚（通常バナー）          | イベント、デイリーミッション、プレミアム通貨 | 一般的なキャラクター獲得に使用する        |
| **アップグレード素材** | キャラクターレベルアップ、才能      | ダンジョン（聖域）、探索                 | メインチームの成長を優先する              |

</Section>

All Firingの世界は、ユニークなボクセルサンドボックスであり、ARPGに新鮮な美学を提供します。エルソードのような2Dスクロールの性質のため、最初はナビゲートが混乱するかもしれませんが、探索は非常にやりがいがあります。隠された宝箱、秘密のエリア、貴重なアップグレード素材がマップ全体に散らばっており、プレイヤーは時間をかけて隅々まで発見することを奨励されます。ジャンプや走りを必要とするプラットフォーム要素は、探索に別のダイナミックな層を追加します。

<Callout type="tip">
メインストーリーを急いで進めないでください。ロック解除する新しいエリアごとに探索に時間を割きましょう。熱心な探索によって見つかる報酬は、キャラクターの育成を大幅に助け、そうでなければ広範囲にわたって収集しなければならないリソースを提供します。
</Callout>

### デイリーミッションとストーリー進行

デイリーミッションとストーリーチャプターの進行は、アップグレード素材、通貨、経験値を含む報酬の一貫した供給源です。ストーリーは物語を提供するだけでなく、重要なゲームメカニクスを徐々に導入し、自然なチュートリアルとして機能します。これらを定期的に完了することで、リソースの安定した流れが確保され、キャラクターの進行が維持されます。

| アクティビティ          | 報酬                                     | メリット                                  |
| :---------------------- | :--------------------------------------- | :---------------------------------------- |
| **デイリーミッション**  | ゴールド、経験値、プレミアム通貨、召喚チケット | 一貫したリソース収入、着実な成長          |
| **ストーリーチャプター** | キャラクターアンロック、素材、通貨       | メカニクスの紹介、主要な進行              |
| **ダンジョン（聖域）**   | 特定のアップグレード素材                 | キャラクターのパワーアップのための目的の収集 |

</Section>
