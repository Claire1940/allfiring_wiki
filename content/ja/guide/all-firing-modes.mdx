### セミオートマチック射撃モード

セミオートマチック（半自動）モードは、トリガーを引くたびに1発の弾丸を発射するのが特徴です。このモードは精度、弾薬の節約、および長距離での命中精度の維持に優れています。プレイヤーは1発ごとにトリガーを引く必要があるため、熟練したマークスマンに最適です。

| 利点 | 欠点 | 理想的なシナリオ |
| :----------------------- | :------------------------------- | :------------------------------------------- |
| **高い命中精度** | 発射速度（ROF）が低い | 長距離戦、ヘッドショット狙い |
| **弾薬の効率性** | 近接戦でのDPSが低い | スナイパーライフル、DMR、精密なピストル射撃 |
| **優れた反動制御** | 精密なトリガー規律が必要 | 初弾、制圧射撃、バースト制御 |

### ランピング射撃モード

ランピングモードは、主に競技シーンで見られるハイブリッドなモードです。最初はセミオートの発射速度で始まりますが、トリガーを一定のリズムで引き続けると、あらかじめ設定された上限まで自動的に発射速度が上昇します。これにより、プレイヤーはフルオートのペナルティを受けることなく高い発射速度を達成でき、かつリズム感のある入力が求められます。

| 特徴 | メリット | 用途 |
| :------------------------ | :---------------------------------- | :----------------------------------------- |
| **段階的な発射速度** | プレイヤーのトリガー速度に適応 | ペイントボール、ROF制限のあるeスポーツ |
| **DPSの向上** | 純粋なセミオートよりも高い火力 | 中距離のデュエル、制圧射撃 |
| **制御されたバースト** | フルオートによる無駄撃ちを回避 | 複数ターゲットの交戦、エリア拒否 |

<Callout type="warning" title="トーナメントの制限">
多くの競技トーナメント、特にペイントボールでは、ランピングモードとその最大秒間発射数（BPS）制限に関する厳格なルールがあることに注意してください。失格を避けるために、常にイベントの規定を確認してください。
</Callout>

### フルオートマチック射撃モード

フルオートマチック（全自動）モードは、トリガーを押し続けている間、連続して射撃を行うことができます。このモードは、圧倒的な弾数で敵を圧倒するように設計されており、近接戦闘において制圧と最大のダメージ出力を提供します。しかし、反動の制御と弾薬管理において大きな課題があります。

| 側面 | 影響 | 最適な使用場面 |
| :---------------------- | :------------------------------------ | :------------------------------------------- |
| **最高の発射速度** | 脅威の迅速な排除 | 近接戦、部屋の掃討、側面攻撃 |
| **高い弾薬消費** | 頻繁なリロードが必要 | アサルトライフル、SMG、LMG |
| **大きな反動** | 遠距離での制御が困難 | 制圧射撃、敵陣の突破 |
| **法的な制限** | 現実のスポーツでは制限が多い | 自宅の庭でのゲーム、特定のモード（PvEなど） |

### 射撃モード使用の最適化

| シナリオ | 推奨モード | 理由 |
| :------------------- | :--------------- | :--------------------------------------------------- |
| **長距離スナイピング** | セミオート | 精度を最大化し、弾の拡散を最小限に抑える。 |
| **CQB（近接戦闘）** | フルオート | ターゲットを迅速に圧倒し、高火力を出す。 |
| **中距離の小競り合い** | ランピング / バースト | コントロールと発射速度を両立し、弾薬を節約。 |
| **制圧射撃** | フルオート | 敵を釘付けにし、味方の前進を支援する。 |
| **弾薬の節約** | セミオート | 一発一発を確実に当て、無駄な乱射を防ぐ。 |
| **ターゲット練習** | セミオート | 一貫したエイムとトリガーコントロールに集中する。 |

  <Step num="3" title="練習場でのテスト">

  <Step num="5" title="ロードアウトの保存">

<Section icon={Shield} title="射撃モードに関する法的および倫理的考慮事項">
**すべての射撃モード**は戦略的な利点を提供しますが、特に競技ゲームやエアソフト、ペイントボールのような現実世界のシミュレーションにおいては、ルールと倫理的な影響を認識しておくことが重要です。

| 側面 | 考慮事項 | 影響 |
| :---------------------- | :-------------------------------------------- | :----------------------------------------- |
| **ゲームルール** | 多くのゲームはランクマッチで特定のモード（フルオートなど）を制限しています。 | ペナルティ、失格、不当な優位性 |
| **フェアプレイ** | エクスプロイトや意図しないメカニクスを使用すると、体験を損なう可能性があります。 | コミュニティからの反発、BAN（禁止） |
| **現実世界の合法性** | フルオート武器は多くの場所で厳格に規制されているか、違法です。 | 法的結果、安全上のリスク |
| **プレイヤーのエチケット** | 非競技的なゲームでは、過度な「乱射」や「オーバーシュート」を避けましょう。 | プレイヤーとしての評判低下、毒性 |

<Callout type="info" title="常にルールを確認する">
公式のイベントや競技マッチに参加する前に、許可されている射撃モードや発射速度の制限について、必ず公式ルールブックを確認してください。これによりフェアプレイが保証され、予期せぬペナルティを防ぐことができます。
</Callout>

</Section>
