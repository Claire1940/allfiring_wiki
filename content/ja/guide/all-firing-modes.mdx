### セミオートマチック射撃モード

セミオートマチック射撃では、トリガーを1回引くごとに1発の弾丸を発射します。このモードは精度と弾薬の節約を優先するため、精度が最優先される交戦に最適です。多くの競技プレイヤーは、予測可能な反動と正確なヘッドショットを狙える能力からセミオートを好みます。これは多くの武器のデフォルトモードであり、マスターすべき基本的なスキルです。

| 特性 | 説明 | 最適な使用例 |
|---|---|---|
| **発射速度** | プレイヤーのトリガースピードによって制御 | 長距離交戦、ヘッドショット、弾薬節約 |
| **精度** | 最高の潜在的精度 | スナイピング、指定射手ライフル（DMR）、ピストル |
| **弾薬** | 最も効率的な弾薬使用 | 持続的な交戦、限られた補給 |
| **反動** | 1発ごとに管理可能 | 制御しやすく、予測可能 |

### フルオートマチック射撃モード

フルオートマチック射撃では、トリガーが引かれている間、またはマガジンが空になるまで、武器が連続して弾丸を発射します。このモードは、近接戦闘や敵を制圧するのに優れています。高いダメージ出力を提供しますが、弾薬も急速に消費し、大きな反動があるため制御が困難になる場合があります。

| 特性 | 説明 | 最適な使用例 |
|---|---|---|
| **発射速度** | 連続的、可能な限り最高 | 近接戦闘、部屋の掃討、制圧 |
| **精度** | 反動により低下 | 腰だめ撃ち、エリア制圧 |
| **弾薬** | 急速な消費 | 節約のためには短いバースト射撃を推奨 |
| **反動** | 高く、制御が困難 | 効果的に管理するには練習が必要 |

<Callout type="warning">
フルオート射撃を無差別に使うと、弾薬がすぐに尽きてしまい、無防備になる可能性があります。制御されたバースト射撃のために、トリガーを軽く引くことを学びましょう。
</Callout>

### バースト射撃モード

バースト射撃は、トリガーを1回引くごとに決められた数の弾丸（通常2〜3発）を発射します。このモードは、セミオートの精度とフルオートの持続的な射撃のバランスを取ろうとします。1発の射撃よりも多くの火力を必要とするが、フルオートよりも散らばりを抑えたい中距離の交戦に効果的です。実世界のM16A4やM4カービンは、しばしば3点バーストモードを備えています。

| 特性 | 説明 | 最適な使用例 |
|---|---|---|
| **発射速度** | 1回の引きで固定バースト | 中距離戦闘、制御された攻撃 |
| **精度** | 良好、フルオートよりも優れる | 複数のターゲットを素早く交戦 |
| **弾薬** | 中程度の消費 | バーストダメージに効率的 |
| **反動** | 中程度、管理可能 | フルオートよりも制御が容易 |

</Section>

### ランピングモード

ランピングモードは、ペイントボールゲームや特定の未来的なシューターでよく見られる動的な射撃システムです。セミオートマチック射撃で始まりますが、一貫したトリガー引き速度（例：毎秒3発）を維持すると、システムは自動的に発射速度をはるかに高い、事前に設定された上限（例：毎秒10.5発）まで「ランプアップ」します。これにより、プレイヤーはトリガーを常に引き続けることなく高い発射速度を達成でき、制御と攻撃性のユニークな融合を提供します。

| 特徴 | 説明 | 戦略的利点 |
|---|---|---|
| **初期射撃** | セミオートマチック | 精度、弾薬節約 |
| **ランプアップ条件** | 一貫したトリガー引き（例：3bps） | 高い発射速度への自然な移行 |
| **最大発射速度** | 事前設定された上限（例：10.5bps） | 持続的な制圧、ダメージ増加 |
| **制御** | プレイヤー主導、ボードアシスト | 単発と高速連射の両方を可能にする |

<Callout type="success">
ランピングモードは、初期の精度と、交戦中に素早く持続的な射撃に移行する能力の両方を重視する攻撃的なプレイヤーに最適です。
</Callout>

<Section icon={Target} title="射撃モードの切り替えをマスターする">
**全射撃モード**を効果的に使用することは、各モードが何をするかを知るだけでなく、戦闘中にシームレスにそれらを切り替えることでもあります。このスキルは、勝利と敗北の分かれ目となり、即座の脅威に武器を適応させ、弾薬の使用を最適化することができます。

<Steps>
  <Step num="1" title="状況を評価する">
    交戦する前に、ターゲットまでの距離、敵の数、現在の弾薬数を素早く評価します。これにより、最も適切な初期射撃モードが決定されます。
  </Step>
  <Step num="2" title="クイックスイッチを練習する">
    射撃モードを切り替えるためのキーバインドまたはボタンに慣れてください。精密な射撃のためのセミオートと、突然の近距離遭遇のためのフルオートの間で素早く切り替える練習をします。
  </Step>
  <Step num="3" title="カバーを効果的に利用する">
    モードを切り替える際に、カバーを有利に利用します。これにより、敵の攻撃にさらされることなく武器を調整するための短い時間を得ることができます。
  </Step>
  <Step num="4" title="交戦を予測する">
    今後の交戦を予測するように努めます。狭い通路に入る場合は、武器をフルオートまたはバーストに事前設定します。長い視線範囲をスキャンしている場合は、セミオートに切り替えます。
  </Step>
  <Step num="5" title="交戦後に分析する">
    各銃撃戦の後、最適な射撃モードを使用したかどうかを振り返ります。何ができたでしょうか？この継続的な学習プロセスは、意思決定を洗練させます。
  </Step>
</Steps>

| シナリオ | 推奨射撃モード | 理由 |
|---|---|---|
| **長距離狙撃** | セミオートマチック | 最大精度、最小反動、弾薬節約 |
| **近接戦闘** | フルオートマチック | 高ダメージ、複数の脅威の迅速な制圧 |
| **中距離小競り合い** | バーストファイア | ダメージと制御のバランス、素早い排除に優れる |
| **待ち伏せ/側面攻撃** | フルオートマチック | 奇襲要素、即座の火力で圧倒 |
| **制圧射撃** | フルオートマチック（制御されたバースト） | 敵を釘付けにし、味方の前進を可能にする |
| **弾薬が少ない** | セミオートマチック | 残りの弾薬を節約し、精密な射撃を優先 |

<Callout type="warning" title="トーナメントの制限">
一部の競技環境、特にペイントボールのようなゲームでは、特定の射撃モード（フルオートや特定の制限を超えるランピングなど）が制限されたり、違法になったりする場合があります。常にトーナメントの規則を確認してください。
</Callout>
