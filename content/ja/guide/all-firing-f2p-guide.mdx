### 効率的な引き直し戦略

All Firingでの引き直しプロセスは非常にユーザーフレンドリーです。ゲストアカウントを使用し、チュートリアルを完了し、最初の召喚を行い、目的のキャラクターが得られなかった場合はアカウントデータを削除することができます。これは通常、1回あたり約8分かかります。

<Steps>
  <Step num="1" title="ゲストとして開始">
    ゲームを起動し、ゲストログインオプションを選択します。これにより、必要に応じてデータを迅速に削除できます。
  </Step>
  <Step num="2" title="チュートリアルを素早く進める">
    可能であれば、会話をスキップし、敵を走り抜けて、より早く召喚エリアに到達します。右上には会話をスキップするボタンがあります。
  </Step>
  <Step num="3" title="メール報酬を受け取る">
    召喚が解除されたら、メールアイコンに移動し、すべての報酬を受け取ります。これにより、通常、数回の10連召喚に十分な通貨が得られます。
  </Step>
  <Step num="4" title="召喚を実行">
    ピックアップバナーに焦点を当てます。特徴的な6つ星キャラクター（金色の輝き）または強力な5つ星キャラクターを狙います。
  </Step>
  <Step num="5" title="決定と削除/リンク">
    満足した場合、アカウントをリンクします。そうでない場合、設定 > アカウント > アカウント削除に移動してプロセスを再開します。
  </Step>
</Steps>

<Callout type="warning" title="引き直し目標">
F2Pプレイヤーとして、引き直し中に現在の**ピックアップキャラクター**を狙いましょう。DPSキャラクターは後で代わりが見つかるかもしれませんが、序盤に強力なDPSがいると、初期の進行が大幅に加速します。
</Callout>

</Section>

### 賢いリソース配分

スターダストやエンベライトのようなプレミアム通貨は貴重です。すぐに使う誘惑に抵抗してください。

| リソースの種類 | 入手方法                   | F2P利用戦略                                   |
| :------------ | :----------------------- | :---------------------------------------- |
| **エンベライト**  | デイリークエスト、ストーリー、イベント、事前登録 | 必要な強力なキャラクターや不可欠な装備が特徴の特定のピックアップバナーのために貯める。 |
| **スターダスト**  | デイリークエスト、ストーリー、イベント、ログインボーナス | 主に召喚に使用。チームを大幅に強化するキャラクターが特徴のバナーのために貯める。 |
| **アップグレード素材** | ダンジョン、クエスト、探索       | まずはコアとなる3人のキャラクターのアップグレードに集中する。メインチームが強くなるまでアップグレードを分散させない。 |
| **コンパニオンセレクター** | 初心者特典（50連）、ピティ（300連） | チームの重要な役割を埋める、またはチームに強力な属性シナジーをもたらすキャラクターのために賢く使用する。 |

### F2Pが避けるべきよくある間違い

| 間違いのカテゴリ     | 説明                                     | F2P対抗戦略                                       |
| :--------------- | :--------------------------------------- | :------------------------------------------------ |
| **キャラクター切り替えの無視** | 戦闘中に1人のキャラクターに固執し、潜在的なコンボや生存性を失う。 | 3人のキャラクターを定期的に切り替えて、コンボを維持し、攻撃を回避し、チームのポテンシャルを最大限に活用する。 |
| **リソースの分散**     | あまりにも多くのキャラクターを同時にアップグレードし、全体的に弱いロスターになる。 | ほとんどのコンテンツを効率的にクリアできるほど強くなるまで、すべてのアップグレード素材を主要な3人のキャラクターに集中させる。 |
| **ストーリーチャプターのスキップ** | 貴重な報酬、チュートリアル、基本的なゲームメカニズムを見逃す。 | 常にメインストーリーチャプターに従う。これらは重要なコンテキスト、リソースを提供し、メカニズムを自然に導入する。 |
| **マップ探索の怠慢**   | レベルを急いで進み、隠された宝箱や珍しい素材を見落とす。 | すべてのボクセルの隅々を探索する時間を取る。隠された報酬はF2Pの進行に不可欠である。 |
| **すべてのバナーでの召喚** | 必要不可欠でないバナーにプレミアム通貨を費やし、リソースの枯渇につながる。 | コアチームに大きな利益をもたらす、または非常に汎用性の高いキャラクターが特徴のピックアップバナーのためにプレミアム通貨を貯める。 |
