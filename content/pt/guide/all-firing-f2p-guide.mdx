Compreender a matemática gacha é vital para manter um pool de recursos saudável. All Firing usa uma animação de invocação estilo tabuleiro de xadrez onde peças douradas significam uma unidade de 6 estrelas e peças roxas significam uma unidade de 5 estrelas.

| :--- | :--- | :--- |
| **6-Estrelas (Ouro)** | 1.2% | ~2.0% |
| **5-Estrelas (Roxo)** | SR Padrão | N/A |
| **Materiais de Upgrade** | Alto | N/A |

O sistema de piedade em All Firing é transparente, mas requer gerenciamento cuidadoso de tokens específicos.

| Recurso | Detalhe da Mecânica |
| :--- | :--- |
| **Piedade Difícil** | 80 Puxadas |
| **Loja de Trocas** | 80 Tokens para um personagem garantido |
| **Seletor de Iniciante** | Escolha única após 50 puxadas |
| **Seletor de Longo Prazo** | Escolha a cada 300 puxadas padrão |

<Callout type="tip" title="Dica Profissional">
Os tokens para o banner de Taxa Aumentada e o banner Padrão são diferentes. Não espere que seu progresso em um se transfira para o outro. Sempre concentre sua moeda premium no banner de Taxa Aumentada para garantir unidades DPS de tempo limitado.

A taxa base de 1,2% é relativamente padrão para gachas modernos, mas a piedade de 80 puxadas é bastante generosa, especialmente porque permite uma troca direta na loja. Isso garante que até mesmo os jogadores F2P mais azarados possam eventualmente garantir suas unidades desejadas.
