### ความสัมพันธ์ธาตุ (Elemental Affinities)

ระบบธาตุใน All Firing เพิ่มเลเยอร์ของกลยุทธ์ ทำให้การต่อสู้น่าสนใจและไม่ซ้ำซาก แต่ละธาตุมีลักษณะเด่นดังนี้:

| ธาตุ | จุดเน้นทั่วไป | ประเภทความเสียหาย | การใช้เชิงกลยุทธ์ |
| :------ | :--------------------- | :---------- | :------------------------------------------- |
| **ไฟ (Fire)** | ระเบิดพลัง, ดาเมจสูง | ไฟ | ดาเมจฉับพลัน (Burst), AoE, การเล่นเชิงรุก |
| **น้ำแข็ง (Ice)** | สโลว์, ควบคุม | น้ำแข็ง | การควบคุมฝูงชน, ดีบัฟ, กลยุทธ์การป้องกัน |
| **ลม (Wind)** | รวดเร็ว, ยืดหยุ่น | ลม | โจมตีต่อเนื่อง, ความคล่องตัว, การต่อคอมโบ |
| **สายฟ้า (Thunder)** | ระเบิดพลังรวดเร็ว, ฉูดฉาด | สายฟ้า | ดาเมจเร็ว, การทำให้มึนงง, สกิลที่มีผลกระทบสูง |
| **ความมืด (Dark)** | ลึกลับ, ทรงพลัง | ความมืด | เอฟเฟกต์พิเศษ, การพลิกสถานการณ์, ดีบัฟ |

### บทบาทการต่อสู้ (Combat Roles)

นอกจากธาตุแล้ว ตัวละครแต่ละตัวยังทำหน้าที่ในบทบาทการต่อสู้ที่เฉพาะเจาะจง ซึ่งสำคัญต่อการสร้างทีมที่สอดประสานกัน บทบาทเหล่านี้มีอิทธิพลต่อความสามารถและวิธีที่พวกเขามีส่วนร่วมในกลยุทธ์โดยรวมของคุณ

| บทบาท | คำอธิบาย | ตัวอย่างการสนับสนุน |
| :----------------- | :--------------------------------------------- | :----------------------------------- |
| **Assault** | เน้นการทำดาเมจหนักใส่ศัตรู | DPS หลัก, ดาเมจระเบิดพลังสูง |
| **Support** | มอบบัฟ, ฮีล หรือดีบัฟให้พันธมิตร | การฮีล, โล่, การทำให้ศัตรูอ่อนแอลง |
| **Coordinated Attack** | ทำงานร่วมกับตัวละครอื่นเพื่อการโจมตีประสาน | การโจมตีต่อเนื่อง, ตัวต่อคอมโบ |
| **Tank** | รับดาเมจ, ปกป้องพันธมิตรในแนวหน้า | ยั่วยุ, ลดดาเมจ, HP สูง |
| **Tactical/Strategy** | ใช้กลไกพิเศษหรือการควบคุมสนามรบ | การปรับเปลี่ยนสภาพแวดล้อม, บัฟพิเศษ |
| **Utility** | มอบทักษะที่หลากหลายสำหรับสถานการณ์ต่างๆ | บัฟเล็กน้อย, ดีบัฟ, ดาเมจเสริม |

### การพัฒนาและเสริมพลังตัวละคร

เพื่อเพิ่มศักยภาพของ **ตัวละคร All Firing** ของคุณให้สูงสุด คุณจะต้องลงทุนในการเติบโตผ่านระบบต่างๆ ซึ่งรวมถึงการเพิ่มเลเวล, การอัปเกรดพรสวรรค์ (Talents), การติดตั้ง Familiar, และการใช้ Relics และ Engravings

<Steps>
  <Step num="1" title="การเพิ่มเลเวลตัวละคร">
    รับประสบการณ์ผ่านการต่อสู้และการดำเนินเนื้อเรื่อง ใช้วัสดุประสบการณ์เพื่อเพิ่มเลเวลตัวละครอย่างรวดเร็ว ปลดล็อกค่าสถานะที่สูงขึ้นและการเข้าถึงพรสวรรค์ใหม่ๆ
  </Step>
  <Step num="2" title="การอัปเกรดแผนผังพรสวรรค์ (Talent Tree)">
    ตัวละครแต่ละตัวมีแผนผังพรสวรรค์ที่ไม่ซ้ำกัน ใช้ทรัพยากร (หาได้จาก Sanctum) เพื่ออัปเกรดความสามารถ เพิ่มประสิทธิภาพการต่อสู้และปลดล็อกสกิลใหม่
  </Step>
  <Step num="3" title="การติดตั้ง Familiar">
    Familiar ทำหน้าที่เป็นสัตว์เลี้ยงที่ช่วยในการต่อสู้ สิ่งสำคัญคือต้องจับคู่ Familiar กับตัวละครที่มี **ประเภทธาตุเดียวกัน** เพื่อเพิ่มดาเมจสูงสุดและได้เปรียบทางธาตุ เช่น Familiar ธาตุน้ำแข็งกับตัวละครธาตุน้ำแข็ง
  </Step>
  <Step num="4" title="การจัดหา Relics (อุปกรณ์)">
    Relics คืออุปกรณ์ของตัวละคร มอบค่าสถานะและโบนัสเซ็ต พยายามเลือกเซ็ต Relic ที่ตรงกับธาตุของตัวละครเสมอเพื่อเปิดใช้งานโบนัสที่ทรงพลัง เช่น เพิ่มดาเมจลมสำหรับตัวละครธาตุลม
  </Step>
  <Step num="5" title="Engravings (การปลุกพลัง)">
    เมื่อคุณได้รับตัวละครซ้ำจากระบบกาชา คุณสามารถใช้เพื่อทำ Engravings ซึ่งจะมอบโบนัสที่สำคัญ เช่น การสะสม Strength Resonance โดยอัตโนมัติเมื่อเข้าสู่การต่อสู้ ช่วยเพิ่มพลังตัวละครให้ดียิ่งขึ้น
  </Step>
</Steps>

### กลยุทธ์การสลับตัวละครที่มีประสิทธิภาพ

| กลยุทธ์ | คำอธิบาย | ผลลัพธ์ |
| :------------------------ | :--------------------------------------------- | :-------------------------------------- |
| **Chain Attacks** | สลับตัวละครเมื่อมีการส่องแสงเพื่อเปิดใช้งานการโจมตีต่อเนื่อง | ดาเมจสูงสุด, คอมโบที่ยาวนานขึ้น |
| **Elemental Burst** | หมุนเวียนตัวละครเพื่อใช้ประโยชน์จากจุดอ่อนธาตุ | กำจัดศัตรูอย่างรวดเร็ว, ประสิทธิภาพสูงขึ้น |
| **Dodge and Counter** | สลับไปใช้ตัวละครที่มีเฟรมอมตะหรือการหลบหลีกที่รวดเร็ว | หลีกเลี่ยงดาเมจ, เตรียมการสวนกลับ |
| **Role Rotation** | ใช้ตัวละคร Support เพื่อบัฟ แล้วสลับเป็น Assault เพื่อทำดาเมจ | การต่อสู้ที่ยั่งยืน, เพิ่มการอยู่รอด |
| **Stamina Management** | สลับตัวละครที่สกิลติดคูลดาวน์หรือสตามิน่าต่ำออก | รักษาความกดดันในการต่อสู้, รักษาเวลาการใช้สกิล |

### การปรับแต่ง Familiar

Familiar เป็นพันธมิตรที่ทรงพลังซึ่งสามารถส่งผลกระทบอย่างมากต่อประสิทธิภาพของตัวละครของคุณ

| แง่มุมของ Familiar | รายละเอียด | ผลกระทบ |
| :-------------- | :-------------------------------------------- | :--------------------------------------- |
| **การจับคู่ธาตุ** | จับคู่ Familiar กับตัวละครธาตุเดียวกัน | เพิ่มดาเมจธาตุและความได้เปรียบสูงสุด |
| **ประเภทดาเมจ** | Familiar สร้างดาเมจธาตุของตัวเอง | เพิ่มเลเยอร์ของพลังโจมตีเสริม |
| **การสนับสนุน** | ช่วยเหลือในการต่อสู้โดยตรง | มอบการสนับสนุนและดาเมจอย่างต่อเนื่อง |

### การเพิ่มประสิทธิภาพ Relic และอุปกรณ์

Relics เป็นแหล่งหลักของอุปกรณ์และโบนัสค่าสถานะ การเลือกและอัปเกรดอย่างมีกลยุทธ์คือกุญแจสำคัญ

| แง่มุมของ Relic | รายละเอียด | ความสำคัญ |
| :-------------- | :----------------------------------------------- | :--------------------------------------- |
| **ค่าสถานะอุปกรณ์** | Relic แต่ละชิ้นมอบโบนัสค่าสถานะต่างๆ | เพิ่มคุณสมบัติของตัวละครโดยตรง |
| **โบนัสเซ็ต** | การสวมใส่อุปกรณ์หลายชิ้นจากเซ็ตเดียวกันมอบโบนัสที่ทรงพลัง | สำคัญสำหรับการสร้างสายเฉพาะทาง (เช่น +ดาเมจลม) |
| **เน้นธาตุ** | ให้ความสำคัญกับ Relics ที่เสริมธาตุหลักของตัวละคร | การเพิ่มพลังที่สอดประสานกัน, ความเชี่ยวชาญเฉพาะสาย |

</Section>
