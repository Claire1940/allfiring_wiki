### โหมดการยิงเซมิออโต้ (Semi-Automatic)

โหมดเซมิออโต้มีลักษณะเด่นคือการยิงเพียงนัดเดียวในแต่ละครั้งที่กดไก โหมดนี้ยอดเยี่ยมสำหรับความแม่นยำ การประหยัดกระสุน และการรักษาความเที่ยงตรงในระยะไกล ผู้เล่นต้องกดไกสำหรับทุกนัดที่ยิง ทำให้เหมาะสำหรับนักแม่นปืนที่มีทักษะ

| ข้อดี | ข้อเสีย | สถานการณ์ที่เหมาะสม |
| :----------------------- | :------------------------------- | :------------------------------------------- |
| **ความแม่นยำสูง** | อัตราการยิง (ROF) ต่ำกว่า | การปะทะระยะไกล, การเล็งที่หัว |
| **ประสิทธิภาพกระสุน** | DPS ช้ากว่าในการต่อสู้ระยะประชิด | ไรเฟิลซุ่มยิง, DMR, การยิงปืนพกที่แม่นยำ |
| **การควบคุมแรงรีคอยล์ดีกว่า** | ต้องอาศัยระเบียบวินัยในการกดไกที่แม่นยำ | การยิงนัดแรก, การยิงขู่, การควบคุมการยิงเป็นชุด |

### โหมดการยิงแรมปิ้ง (Ramping)

โหมดแรมปิ้งเป็นโหมดไฮบริดที่มักพบในการตั้งค่าแบบแข่งขัน เริ่มต้นด้วยอัตราการยิงแบบเซมิออโต้ แต่จะเพิ่มอัตราการยิงโดยอัตโนมัติไปยังขีดจำกัดที่กำหนดไว้หากกดไกอย่างต่อเนื่อง สิ่งนี้ช่วยให้ผู้เล่นได้รับอัตราการยิงที่สูงโดยไม่ต้องรับบทลงโทษแบบฟูลออโต้ ในขณะที่ยังคงต้องใช้จังหวะในการกดไกอยู่บ้าง

| คุณลักษณะ | ประโยชน์ | การใช้งาน |
| :------------------------ | :---------------------------------- | :----------------------------------------- |
| **อัตราการยิงแบบก้าวหน้า** | ปรับตามความเร็วในการกดไกของผู้เล่น | เพนต์บอล, eSports ที่มีการจำกัดอัตราการยิง |
| **DPS ที่เพิ่มขึ้น** | พลังทำลายสูงกว่าเซมิออโต้แบบบริสุทธิ์ | การดวลระยะกลาง, การยิงกดดัน |
| **การยิงเป็นชุดที่ควบคุมได้** | หลีกเลี่ยงการสิ้นเปลืองแบบฟูลออโต้ | การปะทะกับศัตรูหลายตัว, การสกัดกั้นพื้นที่ |

<Callout type="warning" title="ข้อจำกัดในการแข่งขัน">
พึงระวังว่าการแข่งขันระดับมืออาชีพหลายรายการ โดยเฉพาะในกีฬาเพนต์บอล มีกฎที่เข้มงวดเกี่ยวกับโหมดแรมปิ้งและขีดจำกัดจำนวนลูกต่อวินาที (BPS) สูงสุด ตรวจสอบกฎระเบียบของงานเสมอเพื่อหลีกเลี่ยงการถูกตัดสิทธิ์
</Callout>

### โหมดการยิงฟูลออโต้ (Full-Automatic)

โหมดฟูลออโต้ช่วยให้สามารถยิงได้อย่างต่อเนื่องตราบเท่าที่ยังกดไกค้างไว้ โหมดนี้ออกแบบมาเพื่อถล่มคู่ต่อสู้ด้วยปริมาณกระสุนที่มหาศาล มอบการกดดันและความเสียหายสูงสุดในการต่อสู้ระยะประชิด อย่างไรก็ตาม โหมดนี้มาพร้อมกับความท้าทายที่สำคัญในเรื่องแรงรีคอยล์และการจัดการกระสุน

| แง่มุม | ผลกระทบ | การใช้งานที่ดีที่สุด |
| :---------------------- | :------------------------------------ | :------------------------------------------- |
| **อัตราการยิงสูงสุด** | กำจัดภัยคุกคามได้อย่างรวดเร็ว | ระยะประชิด, การเคลียร์ห้อง, การโอบล้อม |
| **การสิ้นเปลืองกระสุนสูง** | ต้องรีโหลดบ่อยครั้ง | ไรเฟิลจู่โจม, SMG, LMG |
| **แรงรีคอยล์มหาศาล** | ควบคุมได้ยากในระยะไกล | การยิงกดดัน, การทำลายแนวป้องกันศัตรู |
| **ข้อจำกัดทางกฎหมาย** | มักถูกจำกัดในกีฬาโลกแห่งความเป็นจริง | เกมในสวนหลังบ้าน, โหมดเกมเฉพาะ (เช่น PvE) |

</Section>

### การเพิ่มประสิทธิภาพการใช้โหมดการยิงของคุณ

| สถานการณ์ | โหมดที่แนะนำ | เหตุผล |
| :------------------- | :--------------- | :--------------------------------------------------- |
| **การซุ่มยิงระยะไกล** | เซมิออโต้ | เพิ่มความแม่นยำสูงสุด ลดการกระจายของกระสุน |
| **การปะทะระยะประชิด (CQB)** | ฟูลออโต้ | ถล่มเป้าหมายอย่างรวดเร็ว ให้ความเสียหายสูง |
| **การปะทะระยะกลาง** | แรมปิ้ง / เบิร์ส | สร้างสมดุลระหว่างการควบคุมและอัตราการยิง ประหยัดกระสุน |
| **การยิงกดดัน** | ฟูลออโต้ | ตรึงศัตรูให้อยู่กับที่ เปิดโอกาสให้เพื่อนร่วมทีมบุก |
| **การประหยัดกระสุน** | เซมิออโต้ | ทุกนัดมีค่า ป้องกันการสาดกระสุนโดยเปล่าประโยชน์ |
| **การฝึกยิงเป้า** | เซมิออโต้ | มุ่งเน้นไปที่การเล็งที่สม่ำเสมอและการควบคุมไกปืน |
