ระบบธาตุเพิ่มชั้นเชิงกลยุทธ์ที่ป้องกันไม่ให้การต่อสู้รู้สึกซ้ำซาก มีธาตุหลักห้าธาตุ แต่ละธาตุมีลักษณะการต่อสู้เฉพาะ:

| ธาตุ | รูปแบบการต่อสู้ | ข้อได้เปรียบหลัก |
| :--- | :--- | :--- |

| **น้ำแข็ง** | ควบคุม / ช้า | แช่แข็งศัตรูและลดความเร็วในการเคลื่อนที่ |
| **ลม** | รวดเร็ว / ยืดหยุ่น | การเคลื่อนที่สูงและศักยภาพคอมโบที่รวดเร็ว |
| **สายฟ้า** | รวดเร็ว / ฉูดฉาด | ความเร็วในการโจมตีสูงและชุดสกิลระเบิด |
| **ความมืด** | ลึกลับ / ประโยชน์ใช้สอย | ดีบัฟที่ทรงพลังและการควบคุมฝูงชน |

นอกจากนี้ ตัวละครยังถูกแบ่งออกเป็นบทบาทเฉพาะ ทีมที่สมดุลมักจะประกอบด้วยตัวสร้างความเสียหายหลักหนึ่งตัวและยูนิตสองตัวที่ให้ประโยชน์ใช้สอยหรือการโจมตีที่ประสานงานกัน

- **แนวหน้า/แทงค์**: ความอยู่รอดสูง อยู่แนวหน้าเพื่อรับความเสียหาย
- **สไตรเกอร์/โจมตี**: เน้นการสร้างความเสียหายสูงสุดในช่วงเวลาสั้นๆ
- **ยุทธวิธี/กลยุทธ์**: ให้บัฟ ควบคุมสนาม หรือดีบัฟศัตรู
- **ยูทิลิตี้/ฮีลเลอร์**: รักษาทีมให้รอดและล้างสถานะที่ไม่ดี
- **การโจมตีที่ประสานงาน**: เชี่ยวชาญในการสร้างความเสียหายจากนอกสนามในขณะที่ยูนิตอื่นทำงานอยู่
