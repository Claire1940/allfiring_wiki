### ระบบความสัมพันธ์ของธาตุ

ระบบธาตุใน All Firing เพิ่มความลึกและกลยุทธ์ให้กับการต่อสู้ มีธาตุหลัก 5 ธาตุ: ไฟ (Fire), น้ำแข็ง (Ice), ลม (Wind), สายฟ้า (Thunder) และมืด (Dark) แต่ละธาตุมีจุดแข็งและจุดอ่อนต่อธาตุอื่นๆ ทำให้การจัดทีมมีความสำคัญอย่างยิ่งสำหรับการเผชิญหน้าที่ท้าทาย การจับคู่ตัวละครกับ Familiars และ Relics ที่เข้ากันตามธาตุจะช่วยเพิ่มประสิทธิภาพได้อย่างมาก

| ธาตุ | จุดเน้นหลัก | เอฟเฟกต์ทั่วไป |
| :------- | :---------------------------- | :------------------------------------------ |
| **ไฟ** | ดาเมจสูง, การระเบิด | การเผาไหม้, ดาเมจวงกว้าง (AoE) |
| **น้ำแข็ง** | การควบคุมฝูงชน, สโลว์ | การแช่แข็ง, การทำให้ศัตรูช้าลง |
| **ลม** | ความคล่องตัว, การต่อสู้ที่ยืดหยุ่น | การโจมตีที่รวดเร็ว, บัฟการเคลื่อนที่ |
| **สายฟ้า** | ดาเมจแบบ Burst, สกิลที่ฉูดฉาด | สตั๊น, ดาเมจเป้าหมายเดี่ยวสูง |
| **มืด** | พลังลึกลับ | ดีบัฟ, เอฟเฟกต์พิเศษที่ทรงพลัง |

### Familiars และ Relics: เสริมคลังแสงของคุณ

Familiars ทำหน้าที่เป็นสัตว์เลี้ยงที่ช่วยในการต่อสู้ มอบดาเมจเพิ่มเติมหรืออรรถประโยชน์ ส่วน Relics คืออุปกรณ์สวมใส่ของคุณ ซึ่งมอบโบนัสค่าสถานะและเอฟเฟกต์เซ็ต ทั้งคู่มีความสำคัญต่อการพัฒนาตัวละครและต้องเลือกอย่างระมัดระวังเพื่อให้เข้ากับธาตุและบทบาทของตัวละคร

| ประเภทไอเทม | หน้าที่ | กลยุทธ์หลัก |
| :-------- | :------------------------------------- | :------------------------------------------ |
| **Familiars** | ช่วยในการต่อสู้, มอบดาเมจธาตุ | จับคู่ธาตุ Familiar ให้ตรงกับธาตุตัวละคร |
| **Relics** | อุปกรณ์ที่มีค่าสถานะและโบนัสเซ็ต | ติดตั้ง Relics ที่เพิ่มธาตุ/บทบาทของตัวละคร |
| **Engravings** | การอัปเกรดตัวละครถาวร | ใช้ตัวละครซ้ำเพื่อรับโบนัสที่ทรงพลัง |
