<Section icon={Target} title="Comment déclencher les cinématiques d'invocation">

Déclencher **toutes les cinématiques d'invocation** n'est pas seulement une question de chance ; cela dépend fortement du type de bannière avec laquelle vous interagissez. Toutes les unités 5 étoiles ne sont pas accompagnées d'un film. Comprendre la distinction entre une unité "Focus" et un "Pity Breaker" est la première étape pour voir ces animations rares.

<Steps>
  <Step num="1" title="Identifier les héros focus">
    Consultez l'onglet "Plus d'infos" d'une bannière pour voir quels héros sont mis en avant. Seuls les héros focus ont généralement des animations de déclenchement.
  </Step>
  <Step num="2" title="Ciblage par couleur">
    Si vous voulez le film d'un personnage spécifique, ne tirez que sur la pierre de couleur associée à ce héros (Rouge, Bleu, Vert ou Incolore).
  </Step>
  <Step num="3" title="Augmenter le taux de pitié">
    Toutes les cinq invocations infructueuses, votre taux d'apparition de 5 étoiles augmente de 0,5 %, ce qui rend plus probable le déclenchement d'une animation.
  </Step>
  <Step num="4" title="Confirmer les paramètres">
    Assurez-vous que l'option "Films d'invocation" est activée dans le menu des paramètres du jeu, sinon le jeu passera directement à la révélation du héros.
  </Step>
</Steps>

<Callout type="tip" title="Efficacité des orbes">
Invoquez toujours en cercles complets (les 5 pierres) si vous débutez, car cela réduit le coût par invocation de 5 orbes à une moyenne de 4.
</Callout>

<Section icon={Shield} title="Gestion des orbes et tableaux de probabilité">

Pour voir **toutes les cinématiques d'invocation**, vous devez gérer vos orbes avec précision. Le système gacha de Fire Emblem Heroes utilise une mécanique de "pitié" qui garantit l'obtention éventuelle d'une unité 5 étoiles, mais atteindre ce point nécessite un investissement significatif.

| Invocations | Taux Focus 5 étoiles | Taux hors Focus | Chance totale 5 étoiles |
| :--- | :--- | :--- | :--- |
| **Début** | 3,0 % | 3,0 % | 6,0 % |
| **10 invocations** | 3,5 % | 3,5 % | 7,0 % |
| **20 invocations** | 4,5 % | 4,5 % | 9,0 % |
| **40 invocations** | 6,5 % | 6,5 % | 13,0 % |
| **Spark (Pitié)** | 100 % (Garanti) | 0,0 % | 100 % |

<Callout type="warning" title="La mécanique Spark">
La plupart des bannières modernes incluent un "Spark" à 40 invocations. Cela vous permet de choisir gratuitement un héros focus, mais notez que ce choix gratuit **ne déclenche pas** le film d'invocation. Vous devez obtenir le héros naturellement via une pierre pour voir l'animation.
</Callout>

Pour maximiser vos chances de voir une séquence de déclenchement, suivez ces règles de ressources :
- **Économisez au moins 160 orbes** : C'est le coût moyen pour atteindre le seuil de 40 invocations du Spark.
- **Évitez les bannières à 3 %** : À moins que vous n'aimiez spécifiquement les personnages, attendez les bannières légendaires à 8 % pour une meilleure valeur.
- **Connexions quotidiennes** : En 2026, les connexions quotidiennes fournissent en moyenne 5 à 7 orbes par jour, la régularité est donc la clé.

</Section>
