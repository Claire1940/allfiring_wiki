### 현재 확률 업 캐릭터 및 일반 배너 유닛

글로벌 출시 시점에 등장하는 확률 업 캐릭터는 대개 강력한 DPS 유닛입니다. 예를 들어, **펠기아**(바람 속성, 고위 여사제)는 뛰어난 유틸리티와 데미지 잠재력 덕분에 공통적인 목표가 됩니다. DPS 유닛은 결국 대체될 수 있지만, 강력한 스타터는 초반 진행을 매우 수월하게 해줍니다.

| 캐릭터 | 속성 | 역할 / 특기 | 리세마라 우선순위 |
| :-------- | :------- | :------------------------ | :-------------- |
| **펠기아** | 바람 | 고위 여사제 / 유틸리티 | **높음** |
| 로젤 | 번개 | 정보 브로커 / 첩보 | 중간 |
| 키리카 | 불 | 뱅가드 / 헤비 해머 | 중간 |
| 나예 | 얼음 | 스트라이커 / 카타나 마스터 | 중간 |
| 허프윈 | 어둠 | 페이스트리 셰프 / 그림자 조작 | 낮음 |
| 시온 | 바람 | 전령 / 숲의 수호자 | 낮음 |
| 일샤 | 얼음 | 서리 백합 / 전략 | 낮음 |
| 라이라 | 바람 | 학자 / 연구원 | 낮음 |

### 소환 확률 및 천장 시스템

올파이어링은 표준적인 가챠 천장 시스템을 갖추고 있습니다. 확률을 이해하면 리세마라 기대치를 관리하는 데 도움이 됩니다.

| 아이템 유형 | 기본 드랍률 | 천장 시스템 |
| :-------- | :------------- | :----------------------------------------- |
| **6성 유닛** | **1.2%** | 특정 배너에서 80회 소환 시 확정 지급 |
| 5성 유닛 | 높음 | 10회 소환 시 자주 등장하지만 영향력은 낮음 |
| 확률 업 6성 | 50/50 확률 | 6성 획득 시 50% 확률로 픽업 캐릭터 등장 |

<Callout type="success">
이 게임은 초보자 기간 동안 **50회 소환 선택권**을 제공하여, 50회 소환 후 동료 한 명을 직접 선택할 수 있게 해줍니다. 이는 부족한 캐릭터를 보충하거나 성장을 위해 중복 캐릭터를 확보하는 데 매우 강력한 도구입니다!
</Callout>

우편 보상을 받은 후에는 일반적으로 일반 배너 10회 소환 1회와 확률 업 배너 10회 소환 2회를 할 수 있는 재화가 모입니다. 펠기아와 같은 최상급 유닛을 얻을 확률이 가장 높은 확률 업 배너에 유료 재화를 우선적으로 투자하세요.
