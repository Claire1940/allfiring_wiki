**비디오 하이라이트 (일반 전투 개요):**
- 공중 콤보를 연장하기 위한 실시간 캐릭터 전환.
- 적을 가두기 위한 복셀 환경 파괴.
- 최대 화면 클리어를 위한 궁극기 타이밍.

| 캐릭터 | 속성 | 역할 / 특기 |
| :--- | :--- | :--- |
| **로젤** | 번개 | 전술 정보 / 지원 |
| **키리카** | 불 | 뱅가드 / 헤비 해머 |
| **나예** | 얼음 | 스트라이커 / 카타나 마스터 |
| **헤르핀** | 어둠 | 그림자 조작 |
| **펠기아** | 바람 | 대사제 / 유틸리티 |

로젤이 필드에 있을 때, 당신의 목표는 무대를 설정하는 것입니다. 그녀의 기술을 사용하여 적에게 "전도" 상태를 적용하세요. 필드가 전기가 통하면 키리카와 같은 불 캐릭터로 전환하여 "과부하" 폭발을 유발하거나, 나예와 같은 얼음 캐릭터로 전환하여 적의 방어구를 파괴하는 "초전도" 디버프를 생성할 수 있습니다.

<Section icon={Monitor} title="MuMuPlayer로 PC에서 플레이하는 방법">

ALLfiring은 모바일 우선 타이틀이지만, 복셀 플랫폼 액션과 실시간 전투의 복잡성 때문에 PC 플레이에 완벽한 후보입니다. MuMuPlayer를 사용하면 정확한 키 매핑이 가능하며, 이는 로젤의 빠른 전환 콤보를 실행하는 데 필수적입니다.

| 기능 | 모바일 경험 | PC (MuMuPlayer) 경험 |
| :--- | :--- | :--- |
| **조작** | 터치스크린 (부정확할 수 있음) | 키보드/마우스 (매우 정확함) |
| **그래픽** | 배터리/발열로 제한됨 | 최대 설정, 60+ FPS |
| **화면 크기** | 작고, UI가 복잡하게 느껴질 수 있음 | 대형 모니터, 더 나은 가시성 |
| **멀티태스킹** | 어려움 | 옆에서 가이드 실행 용이 |

**PC에 ALLfiring 설치 단계:**

1.  **MuMuPlayer 다운로드**: 공식 사이트를 방문하여 Windows 또는 Mac용 최신 버전을 설치하세요.
2.  **구글 로그인**: 앱 센터에 접속하기 위해 설정을 완료하세요.
3.  **ALLfiring 검색**: 게임을 찾아 설치를 클릭하세요. 해당 지역에 없는 경우, APK 파일을 수동으로 설치할 수 있습니다.
4.  **매핑 구성**: 캐릭터 1, 2, 3의 키를 쉽게 접근할 수 있는 버튼(예: 1, 2, 3 또는 Q, W, E)으로 설정하세요.
5.  **게임 시작**: 홈 화면에서 게임을 실행하고 향상된 성능을 즐기세요.

<Callout type="success" title="성능 향상">

</Callout>
